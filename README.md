# Stock Analysis Project

## Overview
This project analyzes stock performance using Python.

## What I did
- Pulled data using yfinance
- Calculated daily returns
- Calculated Sharpe Ratio
- Built a portfolio
- Analyzed drawdowns
- Visualized performance with graphs

## Technologies
- Python
- Pandas
- Matplotlib
- yfinance

## Stocks analyzed
- Apple (AAPL)
- Microsoft (MSFT)
- S&P 500 (^GSPC)

## Key Metrics
- Annual Return
- Volatility
- Sharpe Ratio
- Maximum Drawdown

## How to run
1. Install libraries:
pip install yfinance pandas matplotlib
2. Run the notebook:
run all cells
