# 📊 Stock Analysis Project

## Overview
This project analyzes stock performance using Python.

## Part 1 – Basic Analysis
- Pulled market data using yfinance  
- Calculated daily returns  
- Calculated Sharpe ratio  
- Built a simple portfolio  
- Analyzed drawdowns  
- Visualized performance  

## Part 2 – Portfolio Optimization
- Simulated 10,000 random portfolios  
- Calculated expected return and volatility  
- Found the optimal portfolio (max Sharpe ratio)  
- Visualized Efficient Frontier  
- Displayed optimal asset allocation  

## Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- yfinance
